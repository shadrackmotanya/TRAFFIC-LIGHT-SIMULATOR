# Traffic Light Simulator

## Description
This is a simple **Traffic Light Simulator** built with **HTML, CSS, and JavaScript**.  
It allows users to **simulate a traffic light** by clicking a button to change the light between **Red, Yellow, and Green**.

## Features
- Simulates a traffic light with three colors
- Changes light on button click
- Each color represents standard traffic rules:
  - 🔴 Red → Stop  
  - 🟡 Yellow → Prepare to Stop  
  - 🟢 Green → Go  

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. User clicks the “Change Light” button.  
2. JavaScript checks the current state.  
3. The corresponding light turns on (Red, Yellow, or Green).  
4. The next click changes to the next light.
